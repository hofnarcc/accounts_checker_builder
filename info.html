Changelog

Version 2:

The main difference between Version 1 and Version 2 is the addition of:

  • Colored and detailed logging using colorama.
  • Support for checking 2-factor authentication (2FA) with a custom valid link.
  • Proxy feature, enabling the import of proxy lists from files and random proxy selection for each account check.
  • Functionality to add the path of chromedriver.exe to the config.

Version 3:

Version 3 enhances Version 2 with the following improvements:

  • Refactored code for better organization and readability.
  • Function to force close Chrome processes, ensuring Chrome is not running beforehand.
  • Ability to select a Chrome profile directly from the GUI.
  • Integration of webdriver_seleniumwire for cases when Selenium alone is insufficient.
  • Enhanced error handling and user feedback.
  • Optimized proxy handling for performance and security.
